使い方ガイド

「開く → 選ぶ → 墨消し → 確認 → 保存」がすべての基本です。まずはクイックスタートを、慣れてきたら各機能の詳しい手順をご覧ください。

0画面の見方

画面上部のツールバーは3段に分かれています。1段目＝選択と墨消しの主作業、2段目＝ページ編集・ズーム・保存、3段目＝受け取ったPDFの安全チェック。左側のサイドバーはページのサムネイル一覧です。

1クイックスタート（最短5ステップ）

  1. PDFを開く

    「開く…」を押すか、ファイルを画面にドラッグ＆ドロップします。

  2. 消したい場所を選ぶ

    文字をなぞる／単語をクリック／四角で囲む。選んだ範囲は青くマークされます。

  3. 「墨（N）」を押す

    選択したN件を、下のデータごと完全に削除します。墨2

  4. 結果を確認する

    消えているか、消し漏れがないかを必ず目視で確認します。

  5. 保存する

    「保存」を押して、別名で書き出します。

ヒント：操作はいつでも Ctrl+Z（元に戻す）／Ctrl+Y（やり直し）で戻せます。安心して試してください。

2開く・取り込む

「開く…」ボタン、またはウィンドウへのドラッグ＆ドロップで開けます。複数のファイル（PDF・Word・画像）をまとめてドロップすると、ファイル名の順に1つのPDFへ連結します。

  1. 1ファイルを開く

    「開く…」→ファイル選択、または1つだけドロップ。

  2. 複数をまとめて取り込む

    複数ファイルを同時にドロップ → ファイル名順に連結。順番はサムネのドラッグで後から入れ替えできます。

ご注意：Word（.docx等）の取り込みには Microsoft Word が必要です。

3選択する（4つの方法）

消したい箇所は、好きな方法で選べます。選択した範囲は青くマークされ、あとでまとめて処理します。

  1. なぞって選択

    テキストをドラッグ。ドラッグ中にライブプレビューで範囲が見えます。選択：文字なぞり

  2. 単語クリック

    消したい単語をクリックでピンポイント選択。

  3. 四角で囲む

    表・図・印影など、テキストでない部分は矩形で囲みます。

  4. キーワード検索から追加

    語を検索してヒット箇所を選択に追加。🔍 キーワードで検索 🔍 追加

  5. 同じ語を一括追加

    直前に選んだ語と同じ言葉を、文書全体からまとめて追加。同語＋

選択をやり直したいとき：「✕」ボタンで選択をすべてクリアできます。

4墨・白・黄で処理する

選択したら、上部ボタンで処理します。かっこ内の数字（N）は現在の選択件数です。

  1. 墨（完全削除）

    下層のテキスト・画像データごと削除。これが「真の墨消し」です。墨0

  2. 白（白地で消す）

    白く塗りつぶして消します（データも削除する真の墨消し・白地）。白0

  3. 黄（強調ハイライト）

    削除ではなく、目立たせるための黄色マーカー（注釈）。黄0

必ずご確認を：墨消し後は、意図した情報が消え、消し漏れがないかをご自身で確認してください。PDFの構造によっては別途チェックが必要な場合があります。

5スキャンPDF（OCR）

文字情報を持たないスキャンPDFを開くと、OCR（日本語＋英語）の実行確認が出ます。「はい」で認識すると、検索・単語クリック・なぞり選択が使えるようになります。処理はすべてPC内で行われます。

  1. OCRを実行

    確認ダイアログで「はい」。初回のみ言語モデルをダウンロードします（以後オフライン）。

  2. ふつうに墨消し

    認識後は通常のPDFと同じ操作で、画像の中の文字も墨消しできます。

6AIに候補を相談する

固有名詞や指示から、AIに墨消し候補を挙げてもらえます。AI連携/固有名詞…

  1. プロンプトを生成・コピー

    カテゴリや指示を入力すると、ChatGPT / Gemini などに貼り付けるための文面が自動でコピーされます。

  2. AIに貼り付けて質問

    お使いのAIに貼り付け、回答（JSON）を受け取ります。

  3. 結果を貼り戻す

    回答をアプリに貼り戻すと、候補が一覧表示。人の目で取捨選択してから墨消しします。

プライバシー：アプリはPDFを外部に送信しません。AIへ渡すのはコピーした内容をあなたが貼り付けたときだけ。しかも渡す本文は「画面に見えている文字だけ」で、隠し文字はAIに読ませません。

7受け取ったPDFの安全対策

相手から届いたPDFには、画面に見えない「隠し文字」が仕込まれていることがあります。ツールバー3段目の機能で、検出・確認・除去ができます。

  1. 開いた瞬間の警告に気づく

    隠し文字を含むPDFを開くと、上部に赤い警告バナーが出ます。

  2. 隠し文字を確認・削除

    透明・白・不可視・半透明の文字を一覧で確認し、中身を見てから削除。

  3. 徹底照合（OCR）で炙り出す

    「画面に見える文字」と「埋め込まれた文字」をOCRで突き合わせ、食い違いを検出します（時間がかかるため実行前に確認が出ます）。

  4. 提出前クリーニング

    隠し文字・文書プロパティ・添付ファイル・JavaScript をまとめて除去。

  5. 画像化して保存（最も確実）

    全ページを画像に焼き付け、隠し情報を原理的に完全除去。相手に渡す最終版に最適です。

8ページ編集・仕上げ

墨消し以外の仕上げも1本で。多くはサムネで複数選択してからまとめて実行できます。

  1. 回転・削除・並べ替え

    サムネをチェックして一括回転／削除。ドラッグで順番を入れ替え。↺ 左↻ 右🗑 ページ削除

  2. 用紙サイズ・閉じ代

    向きを保ったままA4・B4等に変更／製本用の余白を追加。用紙サイズ閉じ代

  3. ページ番号・スタンプ

    通し番号を一括付与／朱印・社外秘などを押印。№ ページ番号スタンプ

  4. プロパティ消去

    作成者などの文書プロパティを削除。プロパティ消去

9保存する

「保存」から書き出します。ファイルサイズを3段階から選べます。

  1. 原本品質（既定）

    画質を変えずそのまま保存（無改変）。

  2. 標準 / 軽量

    メール添付などサイズを抑えたいときに選択。

  3. 画像化して保存

    隠し情報まで完全に除去したい最終版はこちら（→ 第7章）。

大切なデータの取り扱い：元ファイルは残し、墨消し版は別名で保存することをおすすめします。

★困ったとき・便利なコツ

  1. 間違えた

    Ctrl+Z で元に戻す、Ctrl+Y でやり直し。すべての操作が対象です。

  2. 細部を確認したい

    Ctrl＋マウスホイールで拡大・縮小できます。

  3. 装飾付きの文字が「同語＋」で拾えない

    星などの記号付き文字はOCRのブレで拾い切れないことがあります。その場合はなぞり選択／四角で個別に墨消ししてください。

  4. 消し漏れが心配

    提出前クリーニング＋「画像化して保存」を併用すると安心です。
